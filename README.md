# FuelCost Tracker

Система для учета и анализа расходов на бензин. Помогает автовладельцам отслеживать затраты на топливо, анализировать эффективность расхода и получать статистику по месяцам.

**Ссылка на рабочий проект:** [https://ваш-логин.pythonanywhere.com](https://ваш-логин.pythonanywhere.com)

## Технологии
- **Backend**: Python 3.10, Django 4.2
- **Database**: SQLite (разработка), PostgreSQL (продакшн)
- **Аналитика**: Pandas, Matplotlib
- **Frontend**: Bootstrap 5, HTML5, CSS3
- **Хостинг**: PythonAnywhere
