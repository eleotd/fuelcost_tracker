# Техническое задание: Система анализа расходов на бензин

## 1. Цель проекта
Разработать веб-сервис для владельцев автомобилей, который помогает отслеживать и анализировать расходы на топливо, вычислять средний расход и оптимизировать затраты.

## 2. Роли пользователей
- **Гость**: Может просматривать публичную статистику, использовать калькулятор расхода
- **Авторизованный пользователь**: Может добавлять свои записи о заправках, просматривать свою статистику, анализировать графики
- **Администратор**: Полный доступ к данным через админ-панель

## 3. Модели данных
1. **Car (Автомобиль)**
   - Пользователь (ForeignKey)
   - Марка
   - Модель
   - Год выпуска
   - Объем двигателя
   - Тип топлива

2. **Refueling (Заправка)**
   - Автомобиль (ForeignKey)
   - Дата заправки
   - Количество литров
   - Цена за литр
   - Общая стоимость
   - Пробег на момент заправки
   - Заправлен полный бак (Boolean)

3. **ExpenseCategory (Категория расходов)**
   - Название
   - Описание

4. **OtherExpense (Прочие расходы)**
   - Автомобиль (ForeignKey)
   - Категория (ForeignKey)
   - Дата расхода
   - Описание
   - Сумма

## 4. Ключевой функционал
* Регистрация и авторизация пользователей
* Добавление автомобилей пользователя
* Ведение журнала заправок с автоматическим расчетом расхода топлива
* Графики динамики расхода топлива и цен
* Анализ средних затрат в месяц
* Сравнение эффективности разных автомобилей пользователя
* Калькулятор стоимости поездки

## 5. Внешние интеграции и аналитика
- **Аналитика**: Использование Pandas для расчета статистики
- **Визуализация**: Matplotlib для построения графиков
- **Интеграция с API**: Получение текущих цен на топливо через открытое API (например, данных с АЗС)
