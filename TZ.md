# Техническое задание: FuelCost Tracker

## 1. Цель проекта
Разработка веб-сервиса для учета и анализа расходов на топливо автомобилистами. Система позволяет отслеживать историю заправок, анализировать расход топлива, контролировать затраты и получать рекомендации по оптимизации расходов на основе статистических данных.

## 2. Роли пользователей
- **Гость**: Просмотр информации о сервисе, возможность регистрации и входа
- **Авторизованный пользователь**: Полный доступ к функционалу:
  - Добавление/удаление автомобилей
  - Ведение истории заправок
  - Просмотр статистики и аналитики
  - Управление профилем
- **Администратор**: Управление через Django Admin

## 3. Модели данных
### 3.1 Car (Автомобиль)
- Пользователь (ForeignKey → User)
- Марка, модель, год выпуска
- Тип топлива (AI-92, AI-95, AI-98, Дизель, Газ)
- Объем двигателя
- Средний расход (л/100км)

### 3.2 Refuel (Заправка)
- Пользователь (ForeignKey → User)
- Автомобиль (ForeignKey → Car)
- Дата заправки
- Пробег (одометр)
- Объем топлива
- Цена за литр
- Общая стоимость (вычисляемое поле)
- Признак полного бака
- Название АЗС, примечания

### 3.3 FuelPrice (Цены на топливо)
- Тип топлива
- Средняя цена
- Регион
- Дата обновления
- Источник данных

## 4. Ключевой функционал
* **Регистрация и аутентификация**: Пользователь создает аккаунт и входит в систему
* **Управление автомобилями**: Добавление, редактирование, удаление автомобилей пользователя
* **Ведение заправок**: Форма для внесения данных о каждой заправке с автоматическим расчетом стоимости
* **История и фильтрация**: Просмотр истории заправок с фильтрацией по автомобилям
* **Аналитика и статистика**:
  - Общие затраты на топливо
  - Средний расход по автомобилям
  - Динамика расходов по месяцам
  - Сравнение с текущими рыночными ценами
* **Расчет расхода**: Автоматический расчет расхода топлива между заправками

## 5. Внешние интеграции и аналитика
- **Интеграция с API**: Получение актуальных цен на топливо (используется mock-данные или публичное API)
- **Аналитика**: Расчет статистических показателей (суммы, средние значения, динамика)
- **Визуализация**: Простые графики и диаграммы для наглядного представления данных

## 6. Технологический стек
- **Backend**: Python 3.10.11, Django 5.2.10
- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript
- **База данных**: SQLite 
- **Деплой**: PythonAnywhere
